.NETアプリケーションからのWebサービス呼び出し

.NET Frameworkで作成するアプリケーションからWebサービスを呼び出す場合、「サービス参照」をプロジェクトに追加する必要がある。
(VSの旧バージョンでは「Web参照」と呼ばれていた。)
尚、.NET Core、.NET 5以降は類似の機構として「WCF Web Service Reference」が使用できる。

参照追加の手順

Visual Studio 2010 / .NET Framework 4.x

  1. 追加する先のプロジェクトを右クリックし、「サービス参照の追加」を選択する。
  2. ダイアログに必要な項目を入力する。

Visual Studio 2022 / .NET 6

  1. 追加する先のプロジェクトを右クリックし、「追加」>「サービス参照」を選択する。
  2. 「WCF Web Service」を選択し、「次へ」を押す。
  3. ダイアログに必要な項目を入力する。
