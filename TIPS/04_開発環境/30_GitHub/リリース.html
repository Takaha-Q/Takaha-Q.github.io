GitHubにおけるリリース

GitHubで管理されるリポジトリの特定リビジョンを対象として、ソースコードに加えて任意のファイル(主にビルド済バイナリ)を公開することができる。 リリースされたファイル、およびソースコードはダウンロードが可能である。

手順

Gitでの操作

  1. リリース対象とするコミットに対して、タグを付与する。
    ただし 4. で新たにタグを作成することも可能であるため、必須ではない。

    2. GitHubでの操作

  2. GitHubにログインし、リリース対象のリポジトリを選択する。
  3. "Releases" > "Create a new release" をクリックする。
  4. "Choose a tag" をクリックし、1. で付与したタグを選択する。

    1. でタグを作成していない場合はここで新たに作成することも可能。
    その場合は作成したいタグ名を "Find or create a new tag" 欄に入力し、"Create new tag:[入力したタグ名] on publish" をクリックする。
  5. タイトル、詳細解説を入力、リリース対象物(ビルド済バイナリなど)を追加して "Publish release" をクリックする。
  6. リリース完了。
