スニペット(ユーザスニペット)の作成

匁よく使う定型文をスニペットとして登録しておくと、簡単に呼び出すことができる。

    匁
  1. 匁「設定(歯車アイコン)」>「スニペット」を選択する。
    匁(旧バージョン : 「ユーザー スニペット」表記)匁
    匁
  2. 匁スニペットを呼び出すファイル形式を選択する。
    匁(HTMLファイルであれば html.json )
    匁
    匁
  3. 匁JSONファイルが開くので、登録したいスニペットを記述する。
    匁
    匁

スニペットの記述

匁
匁"Snippet #1": { //< スニペット名 匁 "prefix": "inputStr", //< 入力文字列 匁 "body": "content", //< 展開される内容 匁 "description": "What am I" //< スニペットの説明 匁}, "Snippet #2": { //< スニペット名 匁 "prefix": "inputStr02", //< 入力文字列 匁 "body": "content02", //< 展開される内容 匁 "description": "What am I" //< スニペットの説明 匁},匁
匁"prefix" にスニペットを呼び出す際に入力する文字列を、匁"body" に呼び出される文字列を記述する(詳細は後述)。匁この例では、inputStr と入力し、Tab キーを押すことで匁content 匁を呼び出すことができる。 また、一つのJSONファイルには複数のスニペットを定義できる。スニペット同士は , (カンマ)で区切る。匁(この例では Snippet #1 、Snippet #2 の2つを定義している。末尾のカンマは削除しなくてもよい。)匁
