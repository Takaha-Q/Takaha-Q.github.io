VS Codeでキーボードマクロを使用する

一連の操作をマクロとして登録しておくと、繰り返し用いることができて便利である。
ただし、キーボードマクロは標準機能には含まれないためプラグインを追加する必要がある。

準備 : 拡張機能のインストール

  1. 設定(歯車アイコン) > 「拡張機能」を開く
  2. 検索欄に「Keyboard Macro」と入力し、表示された「Keyboard Macro Beta」をインストールする

https://marketplace.visualstudio.com/items?itemName=tshino.kb-macroからもインストール可能。

マクロの記録と再生

  • 記録(開始・停止) : Ctrl + Alt + R
  • 再生 : Ctrl + Alt + P
